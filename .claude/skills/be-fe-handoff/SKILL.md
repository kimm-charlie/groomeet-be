# FE Handoff

## Overview
API 변경사항을 프론트엔드 팀에 전달하기 위한 문서 작성 규칙을 정의한다.

## 핵심 규칙
- API 변경사항을 문서화하여 프론트엔드 팀과 공유한다.

- 상세 규칙은 `references/fe-handoff-detail.md`를 확인한다.
